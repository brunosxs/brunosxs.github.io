Update to SMRT-Godot

BrunoSXS published on January 27, 2017
1 min, 94 words

Just updated SMRT-Godot with some bugfixes and changes after some months compiling feedback.

What is SMRT-Godot?
It is a dialog system for your games, written as a plugin for godot that enables you to create classic dialog boxes with a lot of personalization.

I have just sent this new version to godot's assetlib, so it is probably not updated there, but on the repo you can find the latest version and some examples.
